# Shop Login Monitor

Automatisches Monitoring von Web-Shop-Logins via GitHub Actions.  
Die konfigurierten Shops werden regelmäßig geprüft. Bei einem fehlgeschlagenen Login wird automatisch ein GitHub Issue erstellt; sobald der Login wieder funktioniert, wird das Issue geschlossen.

## Zweck

Dieses Repository dient der automatischen Überwachung von B2B-Shop-Zugängen, um Ausfälle frühzeitig zu erkennen und zu melden – ohne manuellen Aufwand.
